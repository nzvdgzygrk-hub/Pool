# Pool Aufbau & Sandfilter-Anleitung

Diese Dateien sind für GitHub Pages vorbereitet.

## Hochladen
1. `index.html` in dein Repository hochladen.
2. Den Ordner `bilder` komplett hochladen.
3. In GitHub Pages als Branch `main` und Ordner `/root` auswählen.
4. Seite öffnen.

## Inhalt
- Pool-Plan
- Stützenanordnung gemäß Herstellerplan
- Schritt-für-Schritt-Inbetriebnahme
- Erklärung der Ventilfunktionen
- Originalbilder aus der Anleitung
